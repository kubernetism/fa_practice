# Purchase Order Edit (Reversal + Re-entry) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship an admin-only "Reverse & Re-enter" action on the Purchases screen that atomically negates an existing purchase's inventory/payables/expenses impact, captures a full before/after audit trail, and opens a pre-filled Create Purchase dialog for correction.

**Architecture:** Append-only reversal (original row is never mutated — a sibling negating row is inserted). All cascade writes happen inside a single `withTransaction` call. Three preflight blockers (FIFO consumed, age > configurable days, paid+reconciled) run before any writes. Role is enforced twice — in the renderer (hide button) and in the IPC handler (reject non-admins). Before/after snapshots are serialised into the existing `audit_logs.oldValues` / `newValues` JSON columns.

**Tech Stack:**
- Main process: Electron + TypeScript, `drizzle-orm` over `better-sqlite3-multiple-ciphers`, `ipcMain.handle` + `withTransaction` + existing `createAuditLog`.
- Renderer: React 18, shadcn/ui (`Dialog`, `Button`, `Textarea`, `Badge`), `lucide-react` icons (`RotateCcw`, `AlertTriangle`).
- Tests: `vitest` with in-memory SQLite via the existing `setupTestDatabase` helper.
- Migrations: `drizzle-kit generate` producing `drizzle/0014_purchase_reversal.sql`.

**Related spec:** `docs/plans/2026-04-14-purchase-order-edit-design.md`

```typescript
import { and, eq, sql } from 'drizzle-orm'
import { getDatabase } from '../db'
import {
  purchases,
  purchaseItems,
  inventoryCostLayers,
  accountPayables,
  businessSettings,
  products,
} from '../db/schema'

export interface ReversalSession {
  userId: number
  role: string
  branchId: number | null
}

export interface CheckReversibleResult {
  allowed: boolean
  blockers: string[]
}

const DEFAULT_MAX_DAYS = 90

export async function checkReversible(
  purchaseId: number,
  session: ReversalSession,
): Promise<CheckReversibleResult> {
  const blockers: string[] = []
  if (session.role !== 'admin') {
    return { allowed: false, blockers: ['Only an admin can reverse a purchase order.'] }
  }

  const db = getDatabase()

  const purchase = await db.query.purchases.findFirst({ where: eq(purchases.id, purchaseId) })
  if (!purchase) {
    return { allowed: false, blockers: [`Purchase ${purchaseId} not found.`] }
  }
  if (purchase.status === 'reversed' || purchase.status === 'cancelled') {
    return { allowed: false, blockers: [`Purchase is already ${purchase.status}.`] }
  }

  const items = await db.select().from(purchaseItems).where(eq(purchaseItems.purchaseId, purchaseId))
  const layers = await db.select().from(inventoryCostLayers).where(eq(inventoryCostLayers.purchaseId, purchaseId))

  for (const item of items) {
    const layerTotal = layers
      .filter((l) => l.productId === item.productId)
      .reduce(
        (acc, l) => acc + ((l.quantity ?? 0) - (l.remainingQty ?? 0)),
        0,
      )
    if (layerTotal > 0) {
      const prod = await db.query.products.findFirst({ where: eq(products.id, item.productId) })
      blockers.push(
        `${prod?.name ?? `Product ${item.productId}`}: ${layerTotal} units already sold — use Stock Adjustment instead.`,
      )
    }
  }

  const settingsRow = await db.query.businessSettings.findFirst()
  const maxDays = settingsRow?.purchaseReversalMaxDays ?? DEFAULT_MAX_DAYS
  const ageDays = Math.floor(
    (Date.now() - new Date(purchase.createdAt).getTime()) / 86_400_000,
  )
  if (ageDays > maxDays) {
    blockers.push(`Purchase is ${ageDays} days old; policy allows only ${maxDays} days.`)
  }

  const payable = await db.query.accountPayables.findFirst({
    where: eq(accountPayables.purchaseId, purchaseId),
  })
  if (payable && payable.status === 'paid') {
    blockers.push('Payable is fully paid and reconciled. Unlink payment before reversing.')
  }

  return { allowed: blockers.length === 0, blockers }
}
```

```typescript
import { withTransaction } from './db-transaction'
import { createAuditLog } from './audit'
import { payablePayments } from '../db/schema'
import { expenses } from '../db/schema'

export interface PrefillDraftItem {
  productId: number
  quantity: number
  unitCost: number
}

export interface PrefillDraft {
  supplierId: number
  branchId: number
  items: PrefillDraftItem[]
  shippingCost: number
  taxAmount: number
  paymentMethod: 'cash' | 'cheque' | 'pay_later'
  notes: string
}

export interface ExecuteReversalResult {
  success: boolean
  error?: string
  reversalPurchaseId?: number
  prefillDraft?: PrefillDraft
}

export async function executeReversal(
  purchaseId: number,
  reason: string,
  session: ReversalSession,
): Promise<ExecuteReversalResult> {
  const pre = await checkReversible(purchaseId, session)
  if (!pre.allowed) {
    return { success: false, error: pre.blockers.join(' ') }
  }
  if (!reason || reason.trim().length < 10) {
    return { success: false, error: 'Reason must be at least 10 characters.' }
  }

  try {
    const result = await withTransaction(async ({ db }) => {
      const original = await db.query.purchases.findFirst({ where: eq(purchases.id, purchaseId) })
      if (!original) throw new Error('Purchase disappeared between preflight and execution')

      const items = await db.select().from(purchaseItems).where(eq(purchaseItems.purchaseId, purchaseId))
      const payable = await db.query.accountPayables.findFirst({ where: eq(accountPayables.purchaseId, purchaseId) })
      const payments = payable
        ? await db.select().from(payablePayments).where(eq(payablePayments.payableId, payable.id))
        : []
      const linkedExpense = await db.query.expenses.findFirst({
        where: payable ? eq(expenses.payableId, payable.id) : sql`1 = 0`,
      })

      const productIds = Array.from(new Set(items.map((i) => i.productId)))
      const productsSnapshot: Record<number, number> = {}
      for (const pid of productIds) {
        const p = await db.query.products.findFirst({ where: eq(products.id, pid) })
        if (p) productsSnapshot[pid] = p.stockQuantity
      }

      const oldValues = {
        purchase: original,
        items,
        payable: payable ?? null,
        payablePayments: payments,
        expense: linkedExpense ?? null,
        productStockSnapshot: productsSnapshot,
      }

      const [sibling] = await db.insert(purchases).values({
        purchaseOrderNumber: `${original.purchaseOrderNumber}-REV`,
        supplierId: original.supplierId,
        branchId: original.branchId,
        userId: session.userId,
        subtotal: -original.subtotal,
        taxAmount: -original.taxAmount,
        shippingCost: -original.shippingCost,
        totalAmount: -original.totalAmount,
        paymentMethod: original.paymentMethod,
        paymentStatus: original.paymentStatus,
        status: 'reversed',
        reversesPurchaseId: original.id,
        reversalReason: reason,
        notes: `Reversal of ${original.purchaseOrderNumber}: ${reason}`,
      }).returning()

      for (const it of items) {
        await db.insert(purchaseItems).values({
          purchaseId: sibling.id,
          productId: it.productId,
          quantity: -it.quantity,
          unitCost: it.unitCost,
          totalCost: -it.totalCost,
          receivedQuantity: -it.receivedQuantity,
        })
      }

      await db.update(purchases)
        .set({ status: 'reversed', reversedByPurchaseId: sibling.id, updatedAt: new Date().toISOString() })
        .where(eq(purchases.id, original.id))

      for (const it of items) {
        const prod = await db.query.products.findFirst({ where: eq(products.id, it.productId) })
        if (!prod) throw new Error(`Product ${it.productId} missing`)
        const newQty = prod.stockQuantity - it.quantity
        if (newQty < 0) {
          throw new Error(`Stock for ${prod.name} would go negative (${prod.stockQuantity} - ${it.quantity}).`)
        }
        await db.update(products).set({ stockQuantity: newQty }).where(eq(products.id, it.productId))
        await db.insert(inventoryCostLayers).values({
          productId: it.productId,
          branchId: original.branchId,
          purchaseId: sibling.id,
          quantity: -it.quantity,
          remainingQty: 0,
          unitCost: it.unitCost,
        })
      }

      if (payable) {
        await db.update(accountPayables)
          .set({ status: 'reversed', remainingAmount: 0, updatedAt: new Date().toISOString() })
          .where(eq(accountPayables.id, payable.id))
        for (const pmt of payments) {
          await db.insert(payablePayments).values({
            payableId: payable.id,
            amount: -pmt.amount,
            paymentMethod: pmt.paymentMethod,
            referenceNumber: pmt.referenceNumber,
            notes: `Reversal of purchase ${original.purchaseOrderNumber}`,
            paidBy: session.userId,
          })
        }
      }

      if (linkedExpense) {
        const [counter] = await db.insert(expenses).values({
          branchId: linkedExpense.branchId,
          userId: session.userId,
          categoryId: linkedExpense.categoryId,
          amount: -linkedExpense.amount,
          description: `Reversal of ${original.purchaseOrderNumber}: ${reason}`,
          paymentMethod: linkedExpense.paymentMethod,
          paymentStatus: 'paid',
          supplierId: linkedExpense.supplierId,
          payableId: linkedExpense.payableId,
        }).returning()
        await db.update(expenses)
          .set({ isReversed: true, reversalExpenseId: counter.id, updatedAt: new Date().toISOString() })
          .where(eq(expenses.id, linkedExpense.id))
      }

      const newValues = {
        reversalPurchaseId: sibling.id,
        reason,
        cascadedPayableId: payable?.id ?? null,
        cascadedExpenseId: linkedExpense?.id ?? null,
      }

      await createAuditLog({
        userId: session.userId,
        branchId: session.branchId,
        action: 'reversal_executed',
        entityType: 'purchase',
        entityId: original.id,
        oldValues,
        newValues,
        description: reason,
      })

      const prefillDraft: PrefillDraft = {
        supplierId: original.supplierId,
        branchId: original.branchId,
        items: items.map((it) => ({ productId: it.productId, quantity: it.quantity, unitCost: it.unitCost })),
        shippingCost: original.shippingCost,
        taxAmount: original.taxAmount,
        paymentMethod: original.paymentMethod,
        notes: `Re-entry of reversed ${original.purchaseOrderNumber}`,
      }

      return { reversalPurchaseId: sibling.id, prefillDraft }
    })

    return { success: true, ...result }
  } catch (err) {
    return { success: false, error: err instanceof Error ? err.message : String(err) }
  }
}
```

```tsx
import { useState, useEffect } from 'react'
import { Dialog, DialogContent, DialogHeader, DialogTitle, DialogFooter } from '@/components/ui/dialog'
import { Button } from '@/components/ui/button'
import { Textarea } from '@/components/ui/textarea'
import { AlertTriangle, CheckCircle2, RotateCcw } from 'lucide-react'
import { formatCurrency } from '@/lib/utils'

interface Purchase {
  id: number
  purchaseOrderNumber: string
  supplierName?: string
  totalAmount: number
  createdAt: string
}

interface Preflight { allowed: boolean; blockers: string[] }

interface Props {
  purchase: Purchase | null
  open: boolean
  onClose: () => void
  onConfirmed: (prefillDraft: Record<string, unknown>) => void
}

export function ReverseDialog({ purchase, open, onClose, onConfirmed }: Props) {
  const [preflight, setPreflight] = useState<Preflight | null>(null)
  const [reason, setReason] = useState('')
  const [submitting, setSubmitting] = useState(false)
  const [error, setError] = useState<string | null>(null)

  useEffect(() => {
    if (!open || !purchase) return
    setPreflight(null)
    setReason('')
    setError(null)
    ;(window as any).api.purchases.checkReversible(purchase.id).then(setPreflight)
  }, [open, purchase])

  const canConfirm =
    preflight?.allowed === true && reason.trim().length >= 10 && !submitting

  const handleConfirm = async () => {
    if (!purchase) return
    setSubmitting(true)
    setError(null)
    const result = await (window as any).api.purchases.reverseAndReenter(purchase.id, reason)
    setSubmitting(false)
    if (!result.success) {
      setError(result.error ?? 'Reversal failed.')
      return
    }
    onConfirmed(result.prefillDraft)
  }

  return (
    <Dialog open={open} onOpenChange={(o) => !o && onClose()}>
      <DialogContent className="max-w-lg">
        <DialogHeader>
          <DialogTitle className="flex items-center gap-2">
            <RotateCcw className="h-4 w-4" /> Reverse Purchase Order
          </DialogTitle>
        </DialogHeader>

        {purchase && (
          <div className="space-y-4">
            <div className="rounded border p-3 text-sm bg-muted/30">
              <div className="font-mono font-medium">{purchase.purchaseOrderNumber}</div>
              <div className="text-muted-foreground">
                {purchase.supplierName ?? 'Supplier'} — {formatCurrency(purchase.totalAmount)}
              </div>
            </div>

            {preflight === null ? (
              <p className="text-xs text-muted-foreground">Checking reversibility…</p>
            ) : preflight.allowed ? (
              <div className="flex items-start gap-2 rounded bg-green-500/10 p-2 text-xs text-green-700 dark:text-green-400">
                <CheckCircle2 className="h-4 w-4 mt-0.5" />
                Ready to reverse.
              </div>
            ) : (
              <div className="space-y-1 rounded bg-destructive/10 p-2 text-xs text-destructive">
                <div className="flex items-center gap-1 font-medium">
                  <AlertTriangle className="h-3 w-3" /> Cannot reverse:
                </div>
                <ul className="ml-5 list-disc">
                  {preflight.blockers.map((b, i) => <li key={i}>{b}</li>)}
                </ul>
              </div>
            )}

            <div className="space-y-1">
              <label className="text-xs font-medium">Reason (required, min 10 characters)</label>
              <Textarea
                value={reason}
                onChange={(e) => setReason(e.target.value)}
                placeholder="Why is this being reversed? Logged permanently."
                rows={3}
                disabled={!preflight?.allowed}
              />
            </div>

            <div className="flex items-start gap-2 rounded border border-amber-500/30 bg-amber-500/5 p-2 text-[11px] text-amber-700 dark:text-amber-400">
              <AlertTriangle className="h-3 w-3 mt-0.5" />
              This will reverse stock, payable, and expense impact. The reversal entry and reason are permanent.
            </div>

            {error && <div className="text-xs text-destructive">{error}</div>}
          </div>
        )}

        <DialogFooter>
          <Button variant="outline" onClick={onClose} disabled={submitting}>Cancel</Button>
          <Button variant="destructive" onClick={handleConfirm} disabled={!canConfirm}>
            Reverse and open re-entry form
          </Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  )
}
```
